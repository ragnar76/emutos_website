EmuTOS News

EmuTOS has the following general advantages over Atari TOS:

  • may be loaded as a program instead of requiring a ROM update
  • available in more languages
  • includes a builtin hard disk driver
  • supports DOS-style FAT16 partitions
  • has dual keyboard support for Greek/Russian languages
  • has user-assignable mouse cursors
  • allows installation of a default file viewer for the desktop

Different versions of EmuTOS have additional advantages over the corresponding versions of Atari TOS:

  • 192K ROM vs Atari TOS v1.x
    • supports menu item shortcuts
    • allows the use of alt-X to open disk X
  • 256K ROM vs Atari TOS 2.x
    • supports additional hardware, including alternate RAM, the MonSTer expansion card, the Magnum RAM expansion card, the ICD real-time clock, and the Nova graphics card adapter
    • provides a builtin command-line interface (console)
    • supports the XBIOS sound extension (Falcon-style audio)
    • supports XHDI
  • 512K ROM vs Atari TOS 3.x
    • supports additional hardware, including the 68060 and Apollo 68080 CPUs, and the ICD real-time clock
    • provides a builtin command-line interface (console)
    • supports the XBIOS sound extension (Falcon-style audio)
    • supports XHDI
  • ARAnyM ROM vs Atari TOS
    • provides ARAnyM-specific benefits, including compilation for the 68040 (for greater performance), the usage of STOP to minimize host processor usage, and builtin 68040 MMU support for aranym-mmu
    • provides a builtin command-line interface (console)
    • supports the XBIOS sound extension (Falcon-style audio)
    • supports XHDI
