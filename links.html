EmuTOS Links

EmuTOS has its project home at SourceForge: https://sourceforge.net/projects/emutos/

But sources are hosted on GitHub: https://github.com/emutos/emutos

See the download links on the Download page.

If you are just curious or would like to help us developing this nice little OS, you are invited to subscribe to our mailing list at: https://sourceforge.net/projects/emutos/lists/emutos-devel

You can also visit and like the official EmuTOS Facebook page.

The following emulators are recommended for running EmuTOS. They are actively developed and supported by their authors.

  • ARAnyM for UNIX, Windows and MacOS X
  • Hatari for UNIX, BeOS, Windows, MacOS X and Android
  • Steem SSE for Windows
  • WinUAE for Windows, to run EmuTOS for Amiga

Older emulators can also run EmuTOS:

  • STonX for UNIX with X11
  • STonC for text terminals (UNIX & Windows console)

Hardware able to run EmuTOS:

  • The FireBee can run EmuTOS for ColdFire
  • The Experiment-S Project uses EmuTOS on the Suska board
  • The MiST board can run EmuTOS
  • The MiSTer board can run EmuTOS
  • The Kiwi board can run a modified version of EmuTOS
  • RaspARI uses a modified version of EmuTOS on Raspberry Pi
