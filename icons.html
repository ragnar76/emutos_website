EmuTOS supports user-loadable icons, as supported by TOS2/3/4. You need a file called EMUICON.RSC in the root of your boot drive, and the file must contain standard desktop monochrome icons. Please note that the order of the icons is not the same as Atari TOS, so if you have an existing Atari-compatible set, you'll need to tweak it. This could be changed, but it would break existing EMUDESK.INF files. Additionally, EmuTOS supports user-loadable cursors. Just like the icons you need a file called EMUCURS.RSC in the root of your boot drive.

EmuTOS comes with 3 different icon and cursors sets: the standard builtin ones, ones from a (modified) TOS3 DESKICON.RSC, and ones from an icon set I found on the net.
