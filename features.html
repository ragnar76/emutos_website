EmuTOS Features

The EmuTOS operating system has the following features:

  • It has a fully TOS-compatible BIOS interface
  • It supports all ST(e)/TT XBIOS calls, plus many Falcon XBIOS calls
  • It supports all ST(e)/TT GEMDOS calls
  • It supports all ST(e)/TT VDI and lineA calls
  • It supports all ST(e) AES calls
  • It is single-tasking
  • It is for a single user
  • It is available in English and ten other languages
  • It is available in many configurations, including those for 192KB, 256KB, and 512KB ROMs
  • You can use two floppy disk drives
  • You can use up to 26 hard disk partitions
  • It supports both Atari-style and DOS-style FAT filesystems
  • Its console is VT52-compatible and supports up to 16 colors
  • It supports all Atari ST(e) and TT video modes, as well as most VIDEL modes
  • It provides an internal minimalistic shell called EmuCON
  • It can be used to load FreeMiNT, and later fVDI and XaAES

EmuTOS supports the following hardware on Atari computers:

  • CPU: m68000, m68008, m68010, m68020, m68030, m68040, m68060, Apollo 68080, ColdFire V4e
  • FPU: SFP004, 68881, 68882, builtin FPU on m68040/m68060
  • Memory controller (ST(e), TT, Falcon)
  • Some third-party add-on memory cards, including Magnum, MonSTer
  • Monitor type detection (mono or not)
  • WD 1772 Floppy disk controller
  • DMA controller
  • ACSI, SCSI, IDE interfaces
  • MFP (both MFPs on the TT)
  • SCC (TT, Falcon)
  • YM 2149
  • ST(e)/TT shifter
  • VIDEL (partially)
  • Nova graphics adapter
  • ACIAs, IKBD protocol
  • MegaST Real Time Clock(RTC), plus many third-party RTCs
  • NVRAM (including RTC)
  • DMA sound
  • Blitter
  • Microwire
Due to size constraints, the 192KB and 256KB ROMs do not support all of the above features and hardware.
