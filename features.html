EmuTOS Features

The EmuTOS operating system aims to be a superset of the original Atari TOS. Like Atari TOS, it is single-user, single-tasking (with limited cooperative multitasking via desk accessories). It is available in many configurations, including those for 192KB, 256KB, and 512KB ROMs, which are suitable as drop-in replacements for TOS in various Atari systems. It still lacks some features of the later versions of Atari TOS, including the AES calls introduced by TOS v3/v4, and support for the Falcon's DSP and 16-bit colour mode. We do expect to implement the missing functionality in future releases.

However, it already surpasses TOS in some areas, by providing:

  • a builtin hard disk driver
  • (in most configurations) a builtin command-line interface (EmuCON2), somewhat like the DOS command line
  • support for ten languages in addition to English
  • (in all ROM configurations) a desktop (EmuDesk) with TOS2-like features

As noted above, EmuTOS does not yet support all of the hardware on original Atari systems. The following is a list of currently unsupported hardware:

  • Falcon DSP
  • 16-bit modes of Falcon Videl

All other standard Atari hardware is supported. In addition, EmuTOS has builtin support for the following hardware, depending on the selected configuration:

  • CPU: Motorola m68020, m68040, m68060; Apollo 68080; ColdFire V4e
  • FPU: builtin FPU on m68040/m68060
  • Some third-party add-on memory cards, including Magnum, MonSTer
  • Nova graphics adapter
  • Many third-party Real Time Clocks
